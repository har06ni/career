Build a complete production-level AI-powered career platform web application similar to the reference design/video.

The product name is "CareerAI".

Goal:
Create a complete SaaS platform where students can create profiles, get AI career recommendations, improve skills, find jobs, connect with mentors, and companies can find and hire students.

The application must be fully functional with frontend, backend, database, authentication, APIs, and responsive UI.
